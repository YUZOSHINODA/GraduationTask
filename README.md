## 仕様書？  
入退出を取る時は教室などの入口に管理者用端末があり、それで入退出管理しているので管理者アカウントにユーザーの入退出、出欠の編集ができる権限を与える  
webから欠席の申請だけ許可する。（ずるな出席を防ぐため）  
出欠の確認が取れていない人にテンプレメールを送付する。（出欠の催促メール）  
出席している人の入退出を取り、欠席の人は入退出をnoneにする
